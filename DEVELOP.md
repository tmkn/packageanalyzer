# Develop

## Setting up the dev environment
```
git clone https://github.com/tmkn/packageanalyzer.git
yarn install
yarn build
```
See [Core Concepts](https://packageanalyzer-docs.vercel.app/docs/core_concepts/provider) for a quick tour around the code

### Watch mode
`yarn dev`
### Build
`yarn build`
## Tests
### Run tests
`yarn test`
### Code Coverage
`yarn test:coverage`
### Watch tests
`yarn test:watch`
